
# 1.7. Anotations (Level 1 - Exercise 2)

## Project Description üìÑ

This Java project solves the following exercise:

"Add some obsolete (deprecated) methods to the child classes, and use the corresponding annotation. Invoke outdated methods from an external class, deleting warnings by the corresponding annotation to be obsolete".

The goal is to understand and practice the use of the @Deprecated annotation in Java, along with managing compiler warnings related to deprecated methods..
